# 할 일 목록 앱 (Todo App)

바닐라 JavaScript로 만든 간단한 할 일 관리 앱입니다.  
별도의 설치나 빌드 과정 없이 `index.html` 파일 하나만 열면 바로 사용할 수 있습니다.

---

## 미리보기

- 할 일 추가 / 완료 체크 / 삭제
- 전체 · 진행중 · 완료 탭으로 필터링
- 새로고침해도 데이터 유지 (localStorage 저장)

---

## 실행 방법

### 방법 1 — 파일 직접 열기
파일 탐색기에서 `index.html`을 더블클릭합니다.

### 방법 2 — VS Code Live Server
1. VS Code에서 **Live Server** 확장을 설치합니다.
2. `index.html`을 우클릭 → **Open with Live Server** 를 선택합니다.

---

## 기능

| 기능 | 설명 |
|---|---|
| 할 일 추가 | 입력창에 텍스트를 쓰고 **추가** 버튼 또는 **Enter** 키를 누릅니다 |
| 완료 처리 | 항목 왼쪽 체크박스를 클릭하면 취소선이 표시됩니다 |
| 개별 삭제 | 항목 오른쪽 **×** 버튼으로 삭제합니다 |
| 일괄 삭제 | 하단 **완료 항목 삭제** 버튼으로 완료된 항목을 모두 지웁니다 |
| 탭 필터 | **전체보기 / 진행중 / 완료** 탭으로 목록을 필터링합니다 |
| 데이터 유지 | 브라우저를 닫거나 새로고침해도 localStorage에 저장되어 유지됩니다 |

---

## 파일 구조

```
todo-app/
├── index.html   # 화면 구조 (HTML)
├── style.css    # 디자인 (CSS)
└── app.js       # 기능 로직 (JavaScript)
```

---

## 기술 스택

- **HTML / CSS / JavaScript** (프레임워크 없음)
- **localStorage** — 브라우저 내장 저장소로 데이터 영속성 처리
- 빌드 도구 없음 — 파일을 그대로 브라우저에서 실행
